### 白话文实现简单的加减法

#### 定义

整数 [name] 等于 [num]

eg:整数 钱包 等于 零

#### 加法

[name] 增加 [num]

eg:钱包 增加 五

#### 减法

[name] 减少 [num]

eg:钱包 减少 五

#### 输出

看看 [name]

eg:看看 钱包

#### 范围

-99 ~ 99









### 如何编译

在本目录下进入控制台

运行指令

```powershell
python compile.py
```





### 如何运行

编译后生成mian.exe

可以直接运行



#### 支持文本读入

比如说我把需要运行的代码写入1.txt

已经编译生成main.exe的前提下

在控制台运行

```powershell
main 1.txt
```

将有输出
